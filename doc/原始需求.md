# TASK

开发一个完整的单页网站（SPA）。

主题：

# 《我只是想做个简单的微博，为什么最后学会了 DDIA》

---

# 核心目标

这不是 PPT。

这不是文档。

这是一个可以用于技术分享的沉浸式滚动网页。

风格类似：

- Apple Keynote
- Stripe
- Linear
- Vercel
- scrollytelling

用户通过滚动页面观看整个故事。

---

# 故事背景

主角是一个普通程序员。

某个周末。

他突然有一个想法：

> “要不我自己做个微博？”

于是开始开发。

随着用户增长。

系统不断出现问题。

主角不断升级架构。

最终一步步走完 DDIA 的全部核心内容。

---

# 最重要要求

不要从技术概念开始讲。

不要出现：

- 什么是索引
- 什么是复制
- 什么是分片
